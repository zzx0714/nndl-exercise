# ReLU神经网络函数拟合报告

## 1. 函数定义

### 1.1 目标函数

本实验选择拟合的目标函数为**二次函数**：

$$y = 0.5x^2 + 0.5x + 1$$

该函数的特点：
- **函数形式**：二阶多项式函数
- **定义域**：实数集 $\mathbb{R}$


### 1.2 理论基础

根据神经网络的通用近似定理（Universal Approximation Theorem），一个具有足够多隐藏单元的两层ReLU神经网络可以以任意精度近似任何可测函数。由于二次函数是连续可微函数，理论上可以被两层ReLU网络完美拟合。

---

## 2. 数据采集

### 2.1 数据生成方法

数据采集通过以下步骤进行：

1. **采样范围**：$x \in [-5, 5]$
2. **采样方式**：均匀随机采样
3. **样本规模**：
   - 训练集：200个样本
   - 测试集：100个样本
4. **噪声添加**：为了增加实验难度并更接近实际数据，添加了高斯噪声
   $$y_{\text{noise}} = y + \epsilon, \quad \epsilon \sim \mathcal{N}(0, 0.05^2)$$

### 2.2 数据预处理

为了提高训练稳定性和收敛速度，对输入数据进行了**归一化处理**：

$$x_{\text{norm}} = \frac{x - x_{\min}}{x_{\max} - x_{\min}} \times 2 - 1$$

其中 $x_{\min} = -5$，$x_{\max} = 5$，处理后的输入值范围在 $[-1, 1]$ 之间。

理由：
- 避免输入值过大导致数值不稳定
- 加快收敛速度
- 提高梯度下降的效率
- 有利于防止梯度爆炸/消失

## 3. 模型描述

### 3.1 网络架构

采用**两层全连接神经网络**，架构如下：

```
输入层 (1个单元)
    ↓
隐藏层 (50个单元, ReLU激活)
    ↓
输出层 (1个单元, 线性激活)
```

### 3.2 前向传播

**第一层**（输入→隐藏层）：
$$\mathbf{z}^{(1)} = \mathbf{x} \mathbf{W}^{(1)} + \mathbf{b}^{(1)}$$

**激活函数**（ReLU）：
$$\mathbf{a}^{(1)} = \text{ReLU}(\mathbf{z}^{(1)}) = \max(0, \mathbf{z}^{(1)})$$

**第二层**（隐藏层→输出层）：
$$\mathbf{\hat{y}} = \mathbf{a}^{(1)} \mathbf{W}^{(2)} + \mathbf{b}^{(2)}$$

其中：
- $\mathbf{W}^{(1)} \in \mathbb{R}^{1 \times 50}$：第一层权重矩阵
- $\mathbf{b}^{(1)} \in \mathbb{R}^{1 \times 50}$：第一层偏置向量
- $\mathbf{W}^{(2)} \in \mathbb{R}^{50 \times 1}$：第二层权重矩阵
- $\mathbf{b}^{(2)} \in \mathbb{R}^{1 \times 1}$：第二层偏置向量

### 3.3 损失函数

采用**均方误差（MSE）**作为损失函数：

$$\mathcal{L} = \frac{1}{2m} \sum_{i=1}^{m} (y_i - \hat{y}_i)^2$$

其中 $m$ 为样本数，$y_i$ 为真实值，$\hat{y}_i$ 为预测值。

### 3.4 反向传播与参数更新

使用**梯度下降法**更新参数：

**输出层梯度**：
$$\frac{\partial \mathcal{L}}{\partial \mathbf{z}^{(2)}} = \hat{y} - y$$

**隐藏层梯度**：
$$\frac{\partial \mathcal{L}}{\partial \mathbf{z}^{(1)}} = \left(\frac{\partial \mathcal{L}}{\partial \mathbf{z}^{(2)}} \mathbf{W}^{(2)T}\right) \odot \text{ReLU}'(\mathbf{z}^{(1)})$$

**参数更新**：
$$\mathbf{W} \leftarrow \mathbf{W} - \alpha \frac{\partial \mathcal{L}}{\partial \mathbf{W}}$$
$$\mathbf{b} \leftarrow \mathbf{b} - \alpha \frac{\partial \mathcal{L}}{\partial \mathbf{b}}$$

其中 $\alpha = 0.01$ 为学习率。


### 3.5 训练参数

| 参数 | 值 |
|------|-----|
| 隐藏层单元数 | 50 |
| 学习率 | 0.01 |
| 批量大小 | 16 |
| 训练轮数（Epoch） | 1000 |
| 优化器 | 小批量梯度下降（Mini-batch SGD） |
| 梯度裁剪范围 | [-1, 1] |

## 4. 拟合效果

### 4.1 定量评估

通过在测试集上的性能指标来评估模型的拟合效果：

| 指标 | 训练集 | 测试集 |
|------|--------|--------|
| MSE（均方误差） | 0.060681 | 0.090223 |
| RMSE（均方根误差） | 0.246335 | 0.300371 |

**结果解析**：
- 训练集MSE为0.061，表明模型在训练数据上的平均误差很小
- 测试集MSE为0.090，略高于训练集，但差异不大，表明模型具有良好的泛化性能
- RMSE数值分别为0.246和0.300，相对于目标函数的取值范围（约0~14），误差比例极小（约2-3%）
- 训练集和测试集性能接近，没有发生显著的过拟合

### 4.2 定性评估

#### 损失曲线分析

训练过程的损失值变化如下：

| Epoch | 损失值 | Epoch | 损失值 |
|-------|--------|-------|--------|
| 100 | 0.971485 | 600 | 0.046446 |
| 200 | 0.095580 | 700 | 0.041746 |
| 300 | 0.069737 | 800 | 0.037972 |
| 400 | 0.056309 | 900 | 0.034001 |
| 500 | 0.050673 | 1000 | 0.030340 |

**观察结果**：

1. **快速下降阶段（Epoch 1-200）**
   - 损失从0.971降至0.096，下降幅度高达90%
   - 表明网络在初期快速学习到目标函数的主要特征
   
2. **稳定优化阶段（Epoch 200-1000）**
   - 损失从0.096逐步降至0.030
   - 下降速度逐渐放缓，表现为对数式收敛
   - 网络继续微调参数，逼近最优解
   
3. **收敛性能**
   - 最终损失值0.030，相对于输出范围极小
   - 没有观察到发散或振荡现象，说明学习率和梯度裁剪有效
   - 收敛过程平稳且可靠

#### 拟合效果观察
从生成的拟合曲线图可以观察到：

1. **拟合准确性**：
   - 模型预测曲线与目标函数曲线基本重合
   - 预测曲线呈现光滑的二次曲线形态

2. **泛化性能**：
   - 训练集MSE（0.061）与测试集MSE（0.090）接近，两者的差异仅为0.029，差异比例约32%，在可接受范围内
   - 表明网络具有良好的泛化能力，未发生严重过拟合
